# Online Feedback and Issue Tracking System

## 📌 Project Overview
The Online Feedback and Issue Tracking System is a web-based application designed for educational institutions. It allows students to submit complaints, issues, or feedback through a simple interface, while administrators can manage and resolve these issues efficiently through a dedicated dashboard.

This system helps improve communication between students and administration by providing a transparent and organized way to handle complaints and feedback.

---

## 🚀 Features

### Student
- Register and Login
- Submit complaints or issues
- View submitted complaints
- Track complaint status
- Send feedback

### Admin
- Admin login
- View all complaints
- Update complaint status
- Manage student feedback
- Dashboard overview of issues
