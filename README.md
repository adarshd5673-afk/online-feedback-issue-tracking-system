# 📌 Online Feedback & Issue Tracking System

## 📖 Overview

This project is a full-stack web application designed for colleges where students can submit feedback or complaints, and administrators can efficiently manage and resolve them through a structured dashboard.

---

## 🚀 Features

### 👨‍🎓 Student Side

* Register and Login system
* Submit feedback or complaints
* View submitted issues

### 🛠️ Admin Side

* Secure admin login
* View all feedback/issues
* Manage and resolve complaints
* Dashboard interface for monitoring

---

## 🏗️ Tech Stack

### Frontend:

* HTML
* CSS
* JavaScript

### Backend:

* Node.js
* Express.js

### Database:

* MongoDB

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/adarshd5673-afk/online-feedback-issue-tracking-system.git
```

2. Navigate to project folder:

```
cd Project2
```

3. Install dependencies:

```
npm install
```

4. Run the server:

```
nodemon index.js
```

or

```
node index.js
```

---

## 🌐 Usage

* Open browser and go to:

```
http://localhost:3000
```

* Register as a student
* Submit feedback
* Login as admin to manage issues
